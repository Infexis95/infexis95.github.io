Topic: Writing Elements


Writing elements are used to display text on the webpage. There are many different writing elements, each with their own purpose. E.g. creating headers, writing paragraphs and headings.


Example code:

<h1> Proper heading words words words <h1>

Additional information:

  • To use a writing element, you need to know what type of text you want displayed on the web.
  • If you want to display important information that the reader's eyes will catch first, you can use the <H1> tag. Headings go from H1 (most important) to H6 (least important). If you want to display secondary information that is not as important to the reader, you can use H6.
  • To write an entire paragraph, you can use the paragraph tag: <p>.
  • If you want to break your text (like enter in word :) ) you can use the... break tag! It's written like this: <br>.
  • You can also make your text bold by using <b> or Italic by using <i>.
  • Want to underline? use <u>.
  • You are in luck, because we have one last formatting tag to go! The emphasis tag! The tag is written like this: <em>.
