Topic: Headers


A header is an element that is used to display intoductory content. It is mainly used in the form of titles in e.g.: Articles, sectioning content and is even used by SEO (Search Engine Optimization)


Example code:

<header> This is a title! </header>

Additional information:

  • To begin the use of a header, create a header tag (see example code).
  • Between the closing angle brackets, put in your header title.
  • You can use CSS to change the font, make it bold, change the color and much more!
