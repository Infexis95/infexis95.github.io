Topic: Anchor


An anchor tag defines a link, which is used to redirect you to another page. A very important attribute in the anchor tag is the href attribute, the href (Hypertext REFerence) attribute is the holds the value (link) to the other page.


Example code:

<a href="hornpub.com"> Click me get redirected! </a>

Additional information:

  • The anchor tag (a tag from now on) is a tag that helps redirecting traffic to another webpage.
  • The href (Hypertext REFerence) is the valueholder for the anchor.
  • Quick security tip: Getting redirected to webpages can be dangerous. If you are aware that an A tag is being used, hover over the link and in the bottom left corner of your browser you can see the link that you'll be redirected to!
